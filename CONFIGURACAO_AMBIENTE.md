# Prompt para Configuração de Ambiente - Sistema WEB

## Instruções
Cole o prompt abaixo na conversa com a IA e preencha os dados entre parênteses com suas informações:

---

## Prompt (Copie e cole aqui)

Estou em uma máquina nova com a pasta do projeto já clonada. Configure meu ambiente Git e GitHub com as seguintes informações:

**Dados Pessoais:**
- Nome: (Seu nome completo)
- Email: (Seu email do GitHub)

**Repositório:**
- URL: https://github.com/007-sistemas/Sistema_WEB_CLOUD_TURSO.git
- Branch padrão: main

**Tarefas:**
1. Configurar Git local (nome, email, credential manager)
2. Inicializar repositório Git (se necessário)
3. Conectar ao repositório remoto
4. Sincronizar com branch main
5. Autenticar com GitHub (teste de push)
6. Instalar dependências do projeto (npm install)
